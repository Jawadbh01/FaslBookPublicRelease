# FaslBook Images Guide

Replace these files in /public to customize the app:

| File | Size | Used for |
|------|------|----------|
| `logo.png` | 512×512 px | Login screen, loading screen |
| `icon-192.png` | 192×192 px | Mobile home screen icon (small) |
| `icon-512.png` | 512×512 px | Mobile home screen icon (large) |
| `banner.png` | 400×300 px | Login page illustration |
| `splash.png` | 390×844 px | PWA splash screen |

## Steps
1. Replace files keeping the same names
2. Run: `npm run build`
3. Deploy to Vercel
